Описание пакетов приложение:

GUI - графический интерфейс
util - утилиты(единственный класс отвечает за подготовку данных для вставки в таблицу GUI)



Описание схемы БД:
________________________________________________________________________________________________________
			Таблица daytype - Таблица справочник:
Поля:
 id -		 идентификационный номер
 abbreviarion -   абревиатура
 typeDesc - 	 описание

			Описание: 
В таблицу включениы только 2 записи относящиеся к календарю(рабочий день, не рабочий день)

________________________________________________________________________________________________________

			Таблица departments 
Поля:
 id - 		идентификационный номер
 abbreviarion -  абревиатура (пример - IT)
 departDesc -	описание (пример - Information Technology)

			Описание:
Таблица нужна для храниения списка отделов предприятия
________________________________________________________________________________________________________

			Таблица employee 
Поля: 
 employee_ID  -	 уникальный идентификатор
 lastName -	 Фамилия
 firstName -	 Имя
 secondName -	 Отчество
 department_ID - код департамента в котором работает
 position_ID - 	 код должности на которой работает
 personalNumber - личный номер

			Описание:
Таблица нужна для хранения данный работников предприятия

________________________________________________________________________________________________________

			Таблица position
Поля:
 id - 		уникальный идентификатор
 posDesc - 	название должности
 department_ID -уникальный идентификатор отдела в котором такая должность

			Описание:
Таблица храния должностей в отделах

________________________________________________________________________________________________________

			Таблица calendar(WorkCalendar)
Поля:
id - 		уникальный идентификатор
day - 		дата
dayType - 	уникальный идентификатор из справочника daytype

			Описание:
Таблица календарь с описанием рабочих и не рабочих дней

________________________________________________________________________________________________________

		Таблица worktimedesc - таблица справочник
Поля:
id -		уникальный идентификатор
abbreviation - абревиатура
worktimedesc - описание 

		Описание:
Таблица с описанием рабочего времени (работал, отпуск, больничный и т.д.)

________________________________________________________________________________________________________

		Таблица worktimesheet
Поля:
id -			уникальный идентификатор
employee_ID -		уникальный идентификатор работника из employee
date_ID - 		уникальный идентификатор даты из calendar
workTimeDesc_ID - 	уникальный идентификатор рабочего времени из worktimedesc

		Описание:
Табель учета рабочего времени работников
